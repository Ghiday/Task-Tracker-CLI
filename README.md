# Task-Tracker-CLI
This is a CLI project its about, creating, deleting, updating your task, you can look into your list what you still need to do or what you already have done.

What do you need.
1. Download Node.js.
2. Use Git clone to copy the code
3. Put it in your code editor.

  How to use it:
  
 Use node index.js add, to add something for example node index.js add going-to-walk-with-my-dog.
 Use node index.js update, to update your task for example node index.js update 1 going-to-walk-with-my-cockroach.
 Use node index.js delete, to delete your task.
 Use node index.js mark-in-progress, to put your status on "in-progress".
 Use node index.js mark-done, to put your status on "done".
 Use node index.js mark-not-done, to put your status on "not-done"
 Use node index.js list, to show your whole list.
 Use node index.js 'list-done' to show the list of which is "done".
 Use node index.js 'list-not-done' to show the list, of which is "not-done"
 Use node index.js 'list-in-progress' to show the list, of which is "in-progress"
